title: Contributing
---

This lesson is developed using [The Carpentries Workbench][workbench].
We welcome all contributions — from fixing typos to suggesting new content.

## How to Contribute

1. **Report an issue:** Open an issue on the
   [GitHub repository](https://github.com/aichemy-hub/rdm-for-chemists/issues)
   describing the problem or suggestion.
2. **Submit a pull request:** Fork the repository, make your changes, and
   submit a pull request. Please ensure your changes build correctly with
   the Workbench before submitting.
3. **Join the discussion:** Use the issue tracker to discuss ideas for new
   episodes, challenges, or improvements.

## Building the Lesson Locally

This lesson uses the [Carpentries Workbench (Sandpaper)][workbench] to build.
See the Workbench documentation for instructions on setting up a local
development environment.

[workbench]: https://carpentries.github.io/sandpaper-docs/
