# Research Data Management for Chemists

A [Carpentries Incubator](https://carpentries-incubator.org/) lesson on
research data management for PhD students and postdocs working in chemistry.

**Rendered lesson:** <https://aichemy-hub.github.io/rdm-for-chemists>

## Overview

This workshop (~2.5–3 hours) covers:

- **Part 1:** General research data management principles — data management
  plans, FAIR data, storage, sharing, licensing, and funder policies.
- **Part 2:** Chemistry-specific data management — reproducibility in
  chemistry, electronic lab notebooks, metadata standards (InChI, JCAMP-DX,
  IUPAC FAIRSpec), chemistry repositories, managing analytical data (NMR, IR,
  MS), and the Physical Sciences Data Infrastructure (PSDI).
